# openmw
My modlist for [OpenMW](https://openmw.org/) (Morrowind engine recreation and improvement). Thanks to all the authors! Feel free to use the "Adult (NSFW)" section or not, please. Contact me at jaume.alcazo@gmail.com

You should use Mod Organizer 2 to install it.
